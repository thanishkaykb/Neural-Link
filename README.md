# NEURAL//LINK 🧠⚡

**Train your brain by decoding and repeating dynamic light patterns.**

Neural Link is a memory-based game where players observe and replicate an evolving sequence of lights. Each level increases in difficulty, testing focus, recall, and reaction speed.

---

## 🎮 Features

* Progressive level-based gameplay
* Dynamic light sequence generation
* Real-time input validation
* Smooth animations and responsive UI
* Game over and restart system

---

## 🛠 Tech Stack

* React
* Tailwind CSS
* JavaScript

## 🎯 How to Play

1. Start the game
2. Watch the light sequence carefully
3. Repeat the pattern using the buttons
4. Progress through increasing levels
5. One wrong move ends the game
